搜索引擎小技巧

  1. "" 双引号表示完全匹配
  2. A + B 连接多个关键词
  3. A -B 搜索结果包含A但是不包含B （注意A后面必须有空格，而-和B之间不能有空格）
    例如： 于谦 -郭德纲 -相声 于是得到明朝于谦的搜索结果
    例如："docker" -百度百科
  4. OR AND
    例如：React OR Vue
  5. NEAR 两个单词离得近
  6. * 文本间填充内容
    例如：习*平
  7. filetype:xxx 搜索对应类型的文件（注意冒号后无空格）
    例如：八次危机 filetype:pdf
  8. site:xxx.com 在某个网站下搜索
    例如：chatgpt site:zhihu.com
  9. link:xxx.com 搜索网页中包含某链接的结果
    例如：link:bilibili.com
  10. intitle:xxx 搜索标题内包含的内容
    例如：intitle:卢克文
  11. related:xxx.com 搜索相关网站
  12. define:xxx 搜索关键词定义
  13. after:2021 搜索2021年以后的内容
    before:2021 搜索2021年以前的内容

Windows 快捷键

Alt

  • Alt + A 截图
  • Alt + Enter 查看文件属性
  • Alt + -> 查看下一个文件夹（同样适用于网页浏览）
  • Alt + <- 查看下一个文件夹（同样适用于网页浏览）
  • Alt + 上箭头 查看当前文件夹的上一层文件夹

窗口操作

  • Alt + Space + C 关闭窗口
  • Alt + Space + X 最大化当前窗口

Win

  • win + E 直接打开电脑资源管理器
  • win + Tab 切换窗口
  • win + X
  • win + D win + M 最小化所有窗口
  • win + R 运行界面

Ctrl

  • Ctrl + Home 光标移动到窗口顶部
  • Ctrl + End 光标移动到窗口底部
  • Ctrl + Shift + Esc 打开任务管理器
